Static sites with Gulp.js and Jekyll
=============================

A template project using:

- Gulp.js
- Jekyll
- Plain HTML
- Sass (3.3)
- Coffeescript
- Bourbon/Neat/Bitters
- Autoprefixer
- BrowserSync
- More to come...

To get started:

1. `git clone https://github.com/kaishin/static-gulp-jekyll.git [project-name]`
2. `cd [project-name]`
3. `npm install`
4. `bundle install`
5. `gulp`
6. Done
